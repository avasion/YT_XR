Ava Kling — K12 STEM educator running an XR class (incl. a 10-day XR summer intensive). Builds interactive web tools *for students*, so prioritizes: code that's easy to teach (vanilla JS, no heavy build step), reliability in a classroom (no API keys/quotas that break mid-lesson), and a consistent design system.

Design system she uses: terracotta `#9C4A24` (hot accent `#FF6B35`), warm pastels on warm off-white, IBM Plex fonts (Serif italic headings, Sans body, Mono labels).

First project: a 3D "connections graph" search engine for MediaPipe/XR/AR/VR/creative-coding videos — built on `3d-force-graph` with a curated dataset (see repo `YT_XR/`).
